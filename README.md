## 📖 Project Overview

The **AI Campus Recruitment System** is an end-to-end intelligent recruitment platform that streamlines campus hiring through AI-powered resume analysis, semantic skill matching, automated candidate ranking, and recruiter dashboards.

Designed with a **microservices-inspired architecture**, the platform integrates modern frontend technologies, scalable backend APIs, AI-powered services, and cloud-hosted infrastructure to deliver a seamless recruitment experience for students and recruiters.

---

## ✨ Key Highlights

- 🤖 AI-powered Resume Analysis
- 📄 PDF & DOCX Resume Parsing
- 🧠 Semantic Skill Extraction using NLP
- 🎯 Intelligent Candidate Ranking
- 💼 Job Posting & Application Management
- 👨‍🎓 Student Dashboard
- 🏢 Recruiter Dashboard
- 🔐 JWT Authentication & Role-Based Authorization
- ☁️ Cloud Deployment with Render
- 🗄️ MongoDB Atlas Integration
- ⚡ RESTful API Architecture
- 📊 Real-Time Recruitment Workflow

---

| 🤖 AI Service | FastAPI |
| 🧠 NLP Engine | spaCy + Sentence Transformers |
